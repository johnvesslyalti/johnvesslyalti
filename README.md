## 🚀 Projects

### [DevNest](https://github.com/johnvesslyalti/dev-nest) — Social Platform Backend
JWT auth with UUID-tracked refresh token rotation, cascade revocation, and SHA-256 IP hashing. Pull-based feed architecture on PostgreSQL. Redis-backed BullMQ async workers. Full CI/CD with E2E test suites.

### [SubTrackr](https://github.com/johnvesslyalti/sub-trackr) — Multi-Tenant Subscription SaaS · [Live](https://johnvesslyalti-subtrackr.vercel.app)
Strict tenant-aware data isolation, OAuth 2.0 auth, hardened session handling. Built for correctness and concurrency safety under shared infrastructure.

### [LiftLog](https://github.com/johnvesslyalti/lift-log) — Fitness Analytics Platform · [Live](https://johnvesslyalti-liftlog.vercel.app)
Normalized PostgreSQL schema with optimized indexing. Aggregation-heavy SQL queries with sub-120ms response latency on training volume and progression metrics.
